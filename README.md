###### PT-BR

# Projeto de Integração .NET Core 5

## JWT Bearer Web API + Dapper

Um exemplo de Web API usando autenticação privada com Dapper e autenticação da API usando JWT Bearer.

Actions:

1. Users
   - SignUp
   - Login
   - Get User List
   - Get Single User
   - User Edit
   - Delete User

2. Auth
   - Generate Token
   - RefreshToken

## UWP

Um exemplo de aplicação Universal Windows Platform para consumir todas as Actions da API.
